# FHx Language Support

Adds syntax highlighting for `.fhx` configuration files.

## Features
- Block-based syntax highlighting
- Key/value assignment highlighting
- String, number, and boolean detection
- Brace-based code folding

## Usage
Open or create a `.fhx` file and syntax highlighting will apply automatically.

## Required Package for Extension Creation
npm install -g vsce
vsce --version
vsce package

## Installation of Extension
code --install-extension fhx-language-0.0.1.vsix
